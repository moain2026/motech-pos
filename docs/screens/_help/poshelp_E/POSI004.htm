   Help Keys

Use: This screen is used to display the keypad keys that can be used in the Sales Invoices screen and the Sales Returns screen. These keys only work by copying the FMRUSW.RES file to the developer's FORMS 60 folder.

How to Use the Screen

In this screen, a column for the Keypad Keys that can be used in the sales invoices and the sales returns appears. While another column appears, showing the function of each key, noting that the user can print reports to help the cashier in using the invoices; since one of the goals of the POS system is facilitating the selling process.

The following report shows all the keys used in the Invoice screen and the function of each key.
