 POS System > System Inputs > 
 POS Customers Group 
 Use: This screen 
allows dividing the cash customers into harmonious groups. Therefore, the user 
can display the various system reports according to this division. Note that the 
customers can be divided according to different criteria such as: the 
geographical location, sales type or any other criterion that complies with the 
organization’s selling policy. Besides, the groups that are added here, in the 
POS Customers Group, appear in the Onyx Pro System and vice 
versa. 
 How to Use the Screen 
 After clicking on the “Add” button, the screen is used as follows: 
 - 
 Group 
Number: Here, the 
system displays the serial number of the group with the ability to modify it. 
 - 
 Group 
Name/ Foreign Name: Here, the group 
name is entered in the system’s default language and in the foreign 
language. 
 - 
 Send 
Message: Ticking this 
flag, means that the messages is active for the group that has been added. 
 - 
 Save: Users press 
this button after completing data entry.
